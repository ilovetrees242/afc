# AFC
AFC is the compiler for my custom compiled programming language called "A flat"
Right now it cant do anything big lol, but mainly I created this project as a last challenge before moving to my goal, of creating my own operating system
A flat programming language is mainly inspired by C, and focuses on minimalism.
Its also inspired by the musical A♭, because A♭ major is my favourite scale
